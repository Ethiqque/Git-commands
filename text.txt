
1. git log с флагом --oneline — Получить сокращённый лог можно с помощью команды. В терминале появятся только первые несколько символов хеша каждого коммита и их комментарии.

2. cat HEAD - Узнать о последнем коммите

cat refs/heads/master

git log 


3. modified+git add = staged - превращение из измененноного в собранный

4. --amend — Исправить последний коммит
git commit --amend --no-edit  - оставит комит лежать до следующего комита, как только появится новый комит, то информация из него пойдет в прошлый.
git commit --amend -m "Новое сообщение" - заменит прошлое сообщение коммита на новое
